### For Each Element:
1. Add case in `DynamicScreen.tsx` switch statement
2. Create dedicated component file if complex (in `components/elements/`)
3. Export from `components/elements/index.ts`
4. Add any required API endpoints
5. Test with admin portal element configuration

### Dependencies to Consider:
- **Maps**: `react-native-maps`
- **Charts**: `react-native-chart-kit` or `victory-native`
- **QR/Barcode**: `react-native-camera` or `expo-barcode-scanner`
- **Signature**: `react-native-signature-canvas`
- **Rich Text**: `react-native-pell-rich-editor`
- **Video**: `react-native-video`
- **Audio**: `react-native-audio-recorder-player`
